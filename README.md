# 作业提交检查工具

一个简单的 Java 命令行工具，用于批量检查学生作业提交情况。通过比对文件夹中的文件名与学生名单，快速统计已交和未交作业的学生。

## 功能

- 支持多个文件夹路径同时检查
- 自动匹配文件名中包含的学生姓名
- 按已交/未交分类输出结果
- 统计已交和未交人数
- 格式化输出，每行显示 5 个姓名

## 使用方法

1. 在 `studentNames` 数组中填入学生姓名
2. 在 `folderPaths` 数组中填入要检查的文件夹路径
3. 编译运行：

```bash
javac src/Main.java -d out
java -cp out Main
```
